# SudoCook

**SudoCook** is a Java-based Command-Line Interface (CLI) application designed to help users manage recipes and kitchen
inventory efficiently. It enables students and home cooks to track their ingredients, discover what they can cook, and
filter recipes based on preparation time and calorie count — all through an intuitive text interface.
